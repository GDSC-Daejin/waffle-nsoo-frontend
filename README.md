# waffle-nsoo-frontend
와플 N수생입니다

# Start
```
1. node js만 설치되어있고 yarn은 설치되어있지 않을때
  npm install yarn
2. yarn이 설치되어있을때
  yarn install 
  yarn add styled-components
  yarn add styled-reset
  yarn add axios
  yarn add react-router-dom
  yarn add @tanstack/react-query
  yarn add firebase

  yarn start
```
